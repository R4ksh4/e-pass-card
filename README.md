# E-Pass Card

电子通行证（e-pass-card）是一款致力于在旧安卓设备上通过极致的前端优化榨干硬件性能的开源通行证展示程序，基于小米4c开发，灵感源自明日方舟的通行证产品，基于HBuilderX+HTML5技术。

## 项目功能

- **5 种光锥效果模式**：从 60fps 陀螺仪驱动到 24Hz 休眠唤醒，再到纯固定动画，按需切换省电
- **光栅卡（Lenticular）**：6 张图叠一起，靠陀螺仪切透明度，晃手机能看到裸眼 3D
- **播放列表**：自动扫本地文件，拖拽排序，记住上次看到哪
- **手势**：单指隐藏 UI，双指长按 3 秒锁定，三指直接屏蔽（防误触）

## 优化

- translate3d 硬件位移，GPU 直接渲染，不重绘像素
- 部分光锥特效静止 5 秒自动停渲染，省电
- 部分光锥特效使用 PNG 贴图替代 CSS 渐变，减少 CPU 计算

## 使用方式

下载项目内的 `e-pass-card.apk` （安装包已经放release里面了）安装即可体验。
